# CareCase UI

A culturally diverse AI-powered simulation platform for clinical training.

## Setup & Installation

1. Activate the Python environment:
   ```bash
   source CCaseUI-env/bin/activate
   ```

2. Run the application:
   ```bash
   python main.py
   ```
